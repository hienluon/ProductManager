Giới thiệu các tính năng: Bảng quản lý, thêm mới, chỉnh sửa các sản phẩm
Thêm mới: 
Trên form thêm mới nhập các trường --> click **NEW** --> Dữ liệu sản phẩm được lưu DB bảng **product**
Chỉnh sửa:
Trên form chỉnh sửa nhập các trường muốn chỉnh sửa --> click **EDIT** --> Dữ liệu sản phẩn được chỉnh sửa và các thông tin chỉnh sửa lưu DB bảng **product**
Xóa:
Trên màn list table các sản phẩm --> Chọn dòng muốn xóa --> click **DELETE** --> Dữ liệu sản phẩm được xóa khỏi DB ở bảng **product**
Reset:
Trên màn hình thêm mới các trường đang nhập --> click **RESET** --> Tất cả các dữ liệu đang nhập ở form được clear
Tìm kiếm:
Tìm kiếm theo tên, nhập tên sản phẩm muốn tìm kiếm ở textbox **SEARCH BY NAME**, tìm kiếm tương đối --> click **SEARCH** trên table list sẽ hiển thi danh sách các sản phẩm 
có tên như đã tìm kiếm
Cấu trúc DB bảng product có các trường:
ID
NAME
PRODUCER
PRICE (USD)
CREATED_AT
